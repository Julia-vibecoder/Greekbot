## 8 принципов метода

### 1. Учить фразы, а не слова
```
❌ Плохо:  solve = решать
✅ Хорошо: I solved this problem yesterday.
```
**Почему:** Слово запоминается с правильным употреблением.

### 2. Новое сразу применять
После показа новой фразы — сразу использовать:
- Recall prompt
- Второй пример
- Mini reuse task

### 3. Повторять мало, но регулярно
```
❌ Плохо:  раз в неделю много
✅ Хорошо: каждый день понемногу
```
**Метафора:** Язык как цветы — поливать понемногу, но регулярно.

### 4. Повторять ВСЛУХ
Не просто смотреть — **читать вслух!**
- Подключается моторная память
- Развивается произношение
- Запоминание происходит "само"

### 5. Ошибка — не провал, а сигнал
```
❌ Плохо:  наказывать за ошибку
✅ Хорошо: отправить слово на повторение
```
Ошибки — естественная часть роста.

### 6. Перевод вторичен
```
Главное: контекст и ситуация
Вторичное: перевод (как fallback)
```
Мозг должен восстанавливать значение через контекст.

### 7. Сначала паттерн, потом правило
```
❌ Плохо:  сначала грамматика, потом примеры
✅ Хорошо: много примеров → потом понимание правила
```

### 8. Практикуем то, что улучшаем
- Чтение → развивает чтение
- Разговор → развивает разговор
- Повторение вслух → развивает fluency

## Реализация в боте

| Принцип | Реализация |
|---------|------------|
| Фразы, не слова | Каждое слово с примером в контексте |
| Сразу применять | 2 разных примера на каждое слово |
| Мало и регулярно | Сессия = 15 слов, spaced repetition |
| Читать вслух | Обязательная кнопка "Прочёл вслух!" |
| Ошибки OK | Нет "неправильных" ответов |
| Контекст главное | Перевод показывается после примера |
| Паттерны | Collocations показывают сочетаемость |
| Active recall | Сначала слово → потом ответ |

## Структура карточки слова

```
┌─────────────────────────────────────┐
│ 🇬🇷 καλημέρα                        │  ← Target word
│                                     │
│ 👀 Показать ответ                   │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│ 🇬🇷 καλημέρα                        │
│ 🇬🇧 good morning                    │  ← Translation
│                                     │
│ 📝 Прочитай вслух:                  │
│ Καλημέρα! Πώς είσαι σήμερα;         │  ← Example 1
│ (Good morning! How are you today?)  │
│                                     │
│ 🔄 Ещё пример:                      │
│ Καλημέρα σου, φίλε μου!             │  ← Example 2
│ (Good morning to you, my friend!)   │
│                                     │
│ 🔗 Коллокации:                      │
│ • καλημέρα σου = good morning to you│  ← Collocations
│ • πολύ καλημέρα = very good morning │     with translations
│ • καλημέρα όλοι = good morning all  │
│                                     │
│ ✅ Прочёл вслух!                    │
└─────────────────────────────────────┘
```

## Сессия обучения

```
/learn
   │
   ├── Выбор темы (food, travel, etc.)
   │
   └── 15 слов в сессии
        │
        ├── [1/15] Слово → Ответ → Прочёл вслух!
        ├── [2/15] ...
        ├── ...
        ├── [10/15] → 🎭 Предложить диалог
        ├── ...
        └── [15/15] → 📊 Итоги сессии
```

## Три очереди слов

| Очередь | Описание | Приоритет |
|---------|----------|-----------|
| **REVIEW** | Слова по расписанию (пора повторить) | ⬆️ Высший |
| **NEW** | Новые слова | ⬇️ Обычный |
| **LEARNED** | Выучены (9/9) | Раз в месяц |

**Логика:** Сначала показываем REVIEW, потом заполняем сессию NEW.

---

## QA критерии для контента

Каждая единица контента должна быть:

- [ ] **Level-appropriate** — A2 грамматика
- [ ] **Semantically correct** — правильный перевод
- [ ] **Natural** — так говорят носители
- [ ] **Non-duplicate** — не повторяет другие примеры
- [ ] **Pedagogically useful** — помогает запомнить
