# 🇬🇷 Greek Vocabulary Telegram Bot

Telegram-бот для изучения греческой лексики уровней Γ2 и B2.

## Словарь

**4 630 слов** в 10 темах:

Каждое слово содержит: греческий, русский, английский, корень, корневое семейство, коллокации, 3 примера с переводом, синонимы, антонимы, мини-диалог, тему, регистр, частотность.

## Как работает

1. `/learn` — выбираешь тему
2. Бот показывает греческое слово — вспоминаешь значение
3. Нажимаешь "Показать ответ" — видишь перевод и пример
4. Выбираешь:
   - **✅ Помню** — знаешь слово, оно больше не появляется
   - **🆕 Новое слово** — не знаешь, слово попадает в цикл 9 повторений
5. Слова из цикла повторений показываются первыми в следующих сессиях

### Спейсд репетишн (метод Брежестовского)

Только для слов, отмеченных как "Новое слово":
- 9 повторений за 23 дня
- Интервалы: день 1, 3, 5, 7, 9, 11, 14, 18, 23
- После 9/9 — слово выучено, повтор через 30 дней

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и инструкция |
| `/learn` | Выбрать тему и начать сессию |
| `/stats` | Прогресс по темам |
| `/reset` | Сбросить весь прогресс |

## Структура проекта

```
├── bot.py                  ← Telegram бот (хендлеры, карточки)
├── database.py             ← SQLite: прогресс, spaced repetition
├── vocab.py                ← Загрузка CSV, формирование сессий
├── agents.json             ← Описание AI-агентов для генерации контента
├── data/
│   └── enriched/
│       └── unified_vocabulary_enriched.csv  ← Словарь (4630 слов)
├── scripts/                ← Скрипты обработки данных
├── requirements.txt
├── Procfile                ← Railway deploy
└── .env                    ← TELEGRAM_BOT_TOKEN (не в git)
```

## Деплой

Хостинг: **Railway** (автодеплой при пуше в `main`).

### Локальный запуск

```bash
pip install -r requirements.txt
echo "TELEGRAM_BOT_TOKEN=your_token" > .env
python bot.py
```
